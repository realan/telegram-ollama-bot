# Telegram Bot + Ollama

MVP Telegram-бот на Python, который принимает текстовые сообщения, отправляет их в локальную модель через Ollama и возвращает ответ пользователю. Каждое сообщение обрабатывается независимо, история диалога не хранится.

## Требования

- Python 3.11+
- Установленный Ollama
- Telegram bot token

## Установка Ollama

1. Установите Ollama по инструкции с официального сайта.
2. Запустите Ollama локально.
3. Загрузите модель:

```bash
ollama pull llama3.1:8b
```

## Настройка проекта

1. Создайте и активируйте виртуальное окружение.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Создайте `.env` на основе `.env.example`.
4. Укажите как минимум:

```env
TELEGRAM_BOT_TOKEN=your_token
OLLAMA_MODEL=llama3.1:8b
```

## Запуск

```bash
python -m app.main
```

## Что умеет бот

- `/start` - приветствие
- `/help` - краткая справка
- текстовое сообщение -> запрос в Ollama -> ответ пользователю

## Ограничения MVP

- только polling
- только текстовые сообщения
- без хранения истории
- без базы данных
- без RAG, tools, voice, image
