# Telegram Bot + Ollama

Telegram-бот на Python, который принимает текстовые сообщения, отправляет их в локальную модель через Ollama и возвращает ответ с учётом истории диалога. История хранится отдельно для каждого пользователя и автоматически сжимается при росте.

## Требования

- Python 3.11+
- Установленный Ollama
- Telegram bot token

## Установка Ollama

1. Установите Ollama по инструкции с официального сайта.
2. Запустите Ollama локально.
3. Загрузите модель:

```bash
ollama pull llama3.1:8b
```

## Настройка проекта

1. Создайте и активируйте виртуальное окружение.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Создайте `.env` на основе `.env.example`.
4. Укажите как минимум:

```env
TELEGRAM_BOT_TOKEN=your_token
OLLAMA_MODEL=llama3.1:8b
```

## Запуск

```bash
python -m app.main
```

## Команды бота

- `/start` — приветствие
- `/help` — краткая справка
- `/clear` — сбросить историю диалога
- любое текстовое сообщение — запрос в Ollama с учётом истории

## Конфигурация (env-переменные)

| Переменная | По умолчанию | Описание |
|---|---|---|
| `TELEGRAM_BOT_TOKEN` | — | Токен бота (обязательно) |
| `OLLAMA_MODEL` | — | Имя модели Ollama (обязательно) |
| `OLLAMA_BASE_URL` | `http://localhost:11434` | Адрес Ollama |
| `OLLAMA_TIMEOUT_SECONDS` | `120` | Таймаут запроса к модели |
| `MAX_HISTORY_MESSAGES` | `10` | Максимум сообщений в контексте |
| `MAX_CONTEXT_CHARS` | `4000` | Максимум символов в контексте |
| `SUMMARY_TRIGGER_MESSAGES` | `5` | Порог для запуска суммаризации |
| `SUMMARY_MAX_CHARS` | `800` | Максимальная длина summary |
| `LOG_MODEL_CONTEXT` | `true` | Логировать контекст перед запросом |

## Ограничения

- только polling
- только текстовые сообщения
- история хранится в памяти процесса (теряется при рестарте)
- без базы данных
- без RAG, tools, voice, image
